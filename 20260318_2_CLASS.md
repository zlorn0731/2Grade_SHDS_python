### 리스트
- 리스트(list) : 하나의 변수에 여러 값을 할당하는 자료형
  - 리스트 = []
  - 튜플 = ()
  - 딕셔너리 = {} → {키:값}
```
| 구분     | 리스트 (list)     | 튜플 (tuple)        | 문자열 (str)      |
| ------ | -------------- | ----------------- | -------------- |
| 표기     | `[ ]`          | `( )`             | `" "` 또는 `' '` |
| 수정 가능  | ✅ 가능 (mutable) | ❌ 불가능 (immutable) | ❌ 불가능          |
| 데이터 타입 | 여러 타입 가능       | 여러 타입 가능          | 문자만            |
| 사용 목적  | 값 변경 필요할 때     | 값 고정할 때           | 텍스트 처리         |
| 속도     | 보통             | 빠름                | 보통             |
```
- 하나의 자료형, 정수형, 실수형 같은 다양한 자료형 포함 가능
#### (예시)
```
colors = ['red', 'blue', 'green']
colors → | 'red' | 'blue' | 'green' |
```

- 증가값(step)
  - 슬라이싱에서는 시작 인덱스와 마지막 인덱스 외에 마지막 자리에 증가값을 넣을 수 있음
  - 증가값은 한 번에 건너뛰는 값의 개수
``` 
증가값 기본 문법 : 변수명[시작 인덱스:마지막 인덱스:증가값]
cities = ['서울', '부산', '인천', '대구', '대전', '광주', '울산', '수원']
cities[::2] ← 2칸 단위로 슬라이싱
→ ['서울', '인천', '대전', '울산']
cities[::-1] ← 역으로 슬라이싱
→ ['수원', '울산', '광주', '대전', '대구', '인천', '부산', '서울']
```

### 튜플(tuple)의 이해
- 튜플(tuple)은 리스트와 같은 개념이지만, 데이터를 변경할 수 없는 자료구조
  - 튜플(tuple) 기본 문법 : variable_name = (element 1, element 2, element 3 ..... element n)
  - () : tuple 기호
```
t = (1, 2, 3)
print(t, t + t, t * 2)
len(t)
- 결과
(1, 2, 3) (1, 2, 3, 1, 2, 3) (1, 2, 3, 1, 2, 3)
3
```
### 튜플의 기초
  - 요소가 하나인 tuple은 ',' 구분 기호를 포함해서 나타낸다. → (10,)
  - 동일한 자료형을 요소로 갖는 tuple → (1, 2, 3), ('hello', 'hi', 'wow')
  - 혼합 자료형을 갖는 tuple → (1, 2, 3, [4, 5, 6], (7, 8), 9, ('apple', 'hi'))
  - Empty tuple(요소가 없는 튜플) → ( )
  - list, 문자열로부터 tuple 생성 가능
    - 문자열 안의 문자들은 tuple의 요소가 됨 → tuple([1, 2, 3]), tuple('python')
   
### 튜플형 연산
- len(), min(), max(), sum() 함수를 tuple에서 사용 가능
- slice 기능도 가능
```
tuple1 = ('Red', 'Green', 'Blue')
len(tuple1) # len 문자열의 문자 개수 변환 : 3

tuple2 = tuple([5, 6, 7, 8])
max(tuple2) # max 최댓값 : 8
min(tuple2) # min 최솟값 : 5
sum(tuple2) # sum 합계 : 26

tuple3 = tuple1 + tuple2
print(tuple3) # ('Red', 'Green', 'Blue', 5, 6, 7, 8)

tuple4 = tuple2 * 3
tuple4 # (5, 6, 7, 8, 5, 6, 7, 8, 5, 6, 7, 8)

tuple1[1] = 'Orange' # 에러 ❌
```

### 딕셔너리(dictionary) 이해
- 딕셔너리(dictionary) : 전화번호부와 같이 키(key)와 값(value)형태로 데이터를 저장하는 자료구조
```
| 학번 | 이름 | 생년월일 | 주소 |
|------|------|---------|------|
| 20230844 | 박지안 | 2004.07.31 | 파주 |
```
- 딕셔너리는 중괄호 {}를 이용하여 키(key)와 값(value)을 요소로 가짐, 키와 값은 :클론으로 구분
- 딕셔너리 변수명 = {key1:value1, key2:value2...} ← dictionary의 요소는 한 쌍의 key:value로 표현
#### (예시)
```
student_info = {20230844:'JIAN', 20230843:'JIJI', 20230832:'JUUA'}
print(student_info[20230844])
- 결과
JIAN

stdDict = {'학번':20230844, '이름':'박지안', '학과':'컴퓨터공학과'}
print(stdDict['학번'])
print(stdDict['이름'])
print(stdDict['학과'])
- 결과
20230844
박지안
컴퓨터공학과
```

- 딕셔너리에 새로운 항목을 추가
  - 변수명[key] = value → key와 value 쌍 추가
#### (예시)
```
stdDict['학년'] = '2학년' # 추가하기
stdDict
- 결과
{'학번': 20230844, '이름': '박지안', '학과': '컴퓨터공학과', '학년': '2학년'}
```
- 딕셔너리의 항목을 삭제하기
  - del 변수명[key] → key와 결합된 구성원소 삭제
#### (예시)
```
del stdDict['학과'] # 삭제하기
stdDict
- 결과
{'학번': 20230844, '이름': '박지안', '학년': '2학년'}
```

- key들로 이루어진 list를 리턴하는 함수
  - 변수명.keys() → key들로 이루어진 list 리턴
#### (예시)
```
stdDict.keys() # key값만 출력하기
- 결과
dict_keys(['학번', '이름', '학년'])
```

- value들로 이루어진 list를 리턴하는 함수
  - 변수명.values() → value들로 이루어진 list 리턴
#### (예시)
```
stdDict.values() # value값만 출력하기
- 결과
dict_values([20230844, '박지안', '2학년'])
```

- key값과 그에 해당하는 value들의 쌍을 tuple로 리턴
  - 변수명.items() → (key, value)로 이루어진 tuple 리턴
#### (예시)
```
stdDict.items() # key와 value 값의 쌍을 tuple로 리턴
- 결과
dict_items([('학번', 20230844), ('이름', '박지안'), ('학년', '2학년')])
```

- tuple에 값이 있는지 없는지 확인
  - key in 변수명
#### (예시)
```
'이름' in stdDict
- 결과
True # 없으면 False
```
