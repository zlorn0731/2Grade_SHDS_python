# 📚 Python

### 조건문의 종류
- if : 조건이 맞는 경우 실행할 명령을 정의
- if ... else ... : 조건에 맞는 경우와 맞지 않는 경우에 대하여 다른 명령을 정의
- if ... elif ... elif ... else : 여러 가지 조건에 따라 다른 명령을 정의

### 비교 연산자
- == | x == y | x, y의 값이 같은가?                  
- != | x != y | x, y의 값이 같지 않은가?             
- >  | x > y  | x값이 y값보다 큰가?                 
- >= | x >= y | x값이 y값 보다 크거나 같은가? (이상) 
- <  | x < y  | x값이 y값보다 작은가?               
- <= | x <= y | x값이 y값 보다 작거나 같은가? (이하) 

#### (예시)
```
x = 10
y = 20
x == y
>> False

x = y
x == y
>> True

x = 100
y = 200
x != y
>> True
x >= y
>> False
```

### 논리 연산자
```
| and | 조건1 and 조건2 | 조건1, 조건2 모두 만족하면 True |
| or  | 조건1 or 조건2  | 조건1, 조건2 둘 중 하나라도 만족하면 True |
| not | not 조건        | 조건이 True이면 False, False면 True |
```
#### (논리 연산자 예시)
```
iq = 120
age = 21
age >= 20 and iq > 130
>> False
→ True and False
>> False

age >= 20 or iq > 130
>> True
→ True or False
>> True

not (age == 20)
>> True
→ not False
>> True
```

### if 조건문
- 조건에 만족하면 명령을 수행
- if 기본 문법
  - 콜론(:) 사용하기
  - 조건문 괄호 사용 방식
  - 들여쓰기 주의하기 → tab
```
if 조건문 :
   명령문1
```
#### (예시)
```
# 90점 이상이면 "장학금 대상자입니다."
# 점수와 상관없이 "수고하셨습니다."
score = int(input("점수를 입력해주세요: "))

if score >= 90:
    print("장학금 대상자입니다.")
else:
    print("장학금 대상자가 아닙니다.")
print("수고하셨습니다.")
- 결과
점수를 입력해주세요:  95
장학금 대상자입니다.
수고하셨습니다.
```

# 현재 시간을 입력 받아 6시간 전의 시간을 알려주는 프로그램
```
- 1 - 6 = -5 + ?
- 1 - 5 = -4 + ?
- 1 - 4 = -3 + ?
- 1 - 3 = -2 + ?
- 1 - 2 = -1 + ?
- 1 - 1 = 0 + ?
- → ? : 24
```
```
now_time = int(input("현재 시간: "))
past_time = now_time - 6

if past_time <= 0:
    past_time += 24
    
print("현재 시간: %d시" % now_time)
print("이전 시간: %d시" % past_time)
- 결과
현재 시간:  6
현재 시간: 6시
이전 시간: 24시
```

### if ~ else ~ 조건문
- 조건문이 참이면 A 수행 거짓이면 B수행
```
조건
 └ True - 명령문 A
 └ False - 명령문 B
```
- if ~ else ~ 조건문 기본 문법
```
if 조건문 :
   명령문 1
else :
   명령문 2
```
#### (예시)
 ```
       [점수 입력 받기]
     True    ⬇️    False
     ⬅️ <점수 >= 90>  ➡️
     ⬇️               ⬇️
장학금 대상자    대상자 아닙니다
     ⬇️               ⬇️
 축하합니다.       다음 기회를..
     ⬇️               ⬇️
     ➡️ 수고하셨습니다 ⬅️
             ⬇️
```
```
score = int(input("점수 입력하세요: "))

if score >= 90:
    print("장학금 대상자")
    print("축하합니다.")
else:
    print("대상자 아닙니다.")
    print("다음 기회를...")
print("수고하셨습니다.")
- 결과 : True
점수 입력하세요:  95
장학금 대상자
축하합니다.
수고하셨습니다.

- 결과 : False
점수 입력하세요:  60
대상자 아닙니다.
다음 기회를...
수고하셨습니다.
```

#### 확인 문제 1
- 문제
  - 정수 n 입력받기(input 함수)
  - 0 이상이면 "입력하신 n은 양수입니다." 라고 안내
  - 0보다 작으면 "입력하신 n은 음수입니다." 라고 안내
  - 마지막에 "프로그램을 종료합니다."라고 메시지 표시
 ```
n = int(input("정수 입력받기: "))

if n >= 0:
    print("입력하신 %d은 양수입니다." % n)
else:
    print("입력하신 %d은 음수입니다." % n)
print("프로그램을 종료합니다.")
- 결과
정수 입력받기:  -5
입력하신 -5은 음수입니다.
프로그램을 종료합니다.
```

### if ~ elif ~ else 조건문
- 여러가지 상황을 고려하는 조건문
```
조건
 └ case1 - 명령문1
 └ case2 - 명령문2
 └ case3 - 명령문3
```
#### (예시)
- 성적에 따라 A, B, C학점 부여하기
  - A : 90이상
  - B : 80이상 (90미만)
  - C : 70이상 (80미만)
 ```
score = int(input("학점을 입력하시오: "))

if score >= 90:
    print("A학점")
elif score >= 80:
    print("B학점")
elif score >= 70:
    print("C학점")
else:
    print("재수강")
- 결과
학점을 입력하시오:  100
A학점
```
