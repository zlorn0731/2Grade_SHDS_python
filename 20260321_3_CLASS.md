# 📚 Python

### 조건문의 종류
- if : 조건이 맞는 경우 실행할 명령을 정의
- if ... else ... : 조건에 맞는 경우와 맞지 않는 경우에 대하여 다른 명령을 정의
- if ... elif ... elif ... else : 여러 가지 조건에 따라 다른 명령을 정의

### 비교 연산자
- == | x == y | x, y의 값이 같은가?                  
- != | x != y | x, y의 값이 같지 않은가?             
- >  | x > y  | x값이 y값보다 큰가?                 
- >= | x >= y | x값이 y값 보다 크거나 같은가? (이상) 
- <  | x < y  | x값이 y값보다 작은가?               
- <= | x <= y | x값이 y값 보다 작거나 같은가? (이하) 

#### (예시)
```
x = 10
y = 20
x == y
>> False

x = y
x == y
>> True

x = 100
y = 200
x != y
>> True
x >= y
>> False
```

### 논리 연산자
```
| and | 조건1 and 조건2 | 조건1, 조건2 모두 만족하면 True |
| or  | 조건1 or 조건2  | 조건1, 조건2 둘 중 하나라도 만족하면 True |
| not | not 조건        | 조건이 True이면 False, False면 True |
```
#### (논리 연산자 예시)
```
iq = 120
age = 21
age >= 20 and iq > 130
>> False
→ True and False
>> False

age >= 20 or iq > 130
>> True
→ True or False
>> True

not (age == 20)
>> True
→ not False
>> True
```

### if 조건문
- 조건에 만족하면 명령을 수행
- if 기본 문법
  - 콜론(:) 사용하기
  - 조건문 괄호 사용 방식
  - 들여쓰기 주의하기 → tab
```
if 조건문 :
   명령문1
```
#### (예시)
```
# 90점 이상이면 "장학금 대상자입니다."
# 점수와 상관없이 "수고하셨습니다."
score = int(input("점수를 입력해주세요: "))

if score >= 90:
    print("장학금 대상자입니다.")
else:
    print("장학금 대상자가 아닙니다.")
print("수고하셨습니다.")
- 결과
점수를 입력해주세요:  95
장학금 대상자입니다.
수고하셨습니다.
```

# 현재 시간을 입력 받아 6시간 전의 시간을 알려주는 프로그램
```
- 1 - 6 = -5 + ?
- 1 - 5 = -4 + ?
- 1 - 4 = -3 + ?
- 1 - 3 = -2 + ?
- 1 - 2 = -1 + ?
- 1 - 1 = 0 + ?
- → ? : 24
```
```
now_time = int(input("현재 시간: "))
past_time = now_time - 6

if past_time <= 0:
    past_time += 24
    
print("현재 시간: %d시" % now_time)
print("이전 시간: %d시" % past_time)
- 결과
현재 시간:  6
현재 시간: 6시
이전 시간: 24시
```

### if ~ else ~ 조건문
- 조건문이 참이면 A 수행 거짓이면 B수행
- 조건
   └ True - 명령문 A
   └ False - 명령문 B
