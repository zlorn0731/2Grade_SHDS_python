# 📚 Python

### 조건문의 종류
- if : 조건이 맞는 경우 실행할 명령을 정의
- if ... else ... : 조건에 맞는 경우와 맞지 않는 경우에 대하여 다른 명령을 정의
- if ... elif ... elif ... else : 여러 가지 조건에 따라 다른 명령을 정의

### 비교 연산자
- == | x == y | x, y의 값이 같은가?                  
- != | x != y | x, y의 값이 같지 않은가?             
- >  | x > y  | x값이 y값보다 큰가?                 
- >= | x >= y | x값이 y값 보다 크거나 같은가? (이상) 
- <  | x < y  | x값이 y값보다 작은가?               
- <= | x <= y | x값이 y값 보다 작거나 같은가? (이하) 

#### (예시)
```
x = 10
y = 20
x == y
>> False

x = y
x == y
>> True

x = 100
y = 200
x != y
>> True
x >= y
>> False
```

### 논리 연산자
```
| and | 조건1 and 조건2 | 조건1, 조건2 모두 만족하면 True |
| or  | 조건1 or 조건2  | 조건1, 조건2 둘 중 하나라도 만족하면 True |
| not | not 조건        | 조건이 True이면 False, False면 True |
```
#### (논리 연산자 예시)
```
iq = 120
age = 21
age >= 20 and iq > 130
>> False
→ True and False
>> False

age >= 20 or iq > 130
>> True
→ True or False
>> True

not (age == 20)
>> True
→ not False
>> True
```

### if 조건문
- 조건에 만족하면 명령을 수행
- if 기본 문법
  - 콜론(:) 사용하기
  - 조건문 괄호 사용 방식
  - 들여쓰기 주의하기 → tab
```
if 조건문 :
   명령문1
```
#### (예시)
```
# 90점 이상이면 "장학금 대상자입니다."
# 점수와 상관없이 "수고하셨습니다."
score = int(input("점수를 입력해주세요: "))

if score >= 90:
    print("장학금 대상자입니다.")
else:
    print("장학금 대상자가 아닙니다.")
print("수고하셨습니다.")
- 결과
점수를 입력해주세요:  95
장학금 대상자입니다.
수고하셨습니다.
```

# 현재 시간을 입력 받아 6시간 전의 시간을 알려주는 프로그램
```
- 1 - 6 = -5 + ?
- 1 - 5 = -4 + ?
- 1 - 4 = -3 + ?
- 1 - 3 = -2 + ?
- 1 - 2 = -1 + ?
- 1 - 1 = 0 + ?
- → ? : 24
```
```
now_time = int(input("현재 시간: "))
past_time = now_time - 6

if past_time <= 0:
    past_time += 24
    
print("현재 시간: %d시" % now_time)
print("이전 시간: %d시" % past_time)
- 결과
현재 시간:  6
현재 시간: 6시
이전 시간: 24시
```

### if ~ else ~ 조건문
- 조건문이 참이면 A 수행 거짓이면 B수행
```
조건
 └ True - 명령문 A
 └ False - 명령문 B
```
- if ~ else ~ 조건문 기본 문법
```
if 조건문 :
   명령문 1
else :
   명령문 2
```
#### (예시)
 ```
       [점수 입력 받기]
     True    ⬇️    False
     ⬅️ <점수 >= 90>  ➡️
     ⬇️               ⬇️
장학금 대상자    대상자 아닙니다
     ⬇️               ⬇️
 축하합니다.       다음 기회를..
     ⬇️               ⬇️
     ➡️ 수고하셨습니다 ⬅️
             ⬇️
```
```
score = int(input("점수 입력하세요: "))

if score >= 90:
    print("장학금 대상자")
    print("축하합니다.")
else:
    print("대상자 아닙니다.")
    print("다음 기회를...")
print("수고하셨습니다.")
- 결과 : True
점수 입력하세요:  95
장학금 대상자
축하합니다.
수고하셨습니다.

- 결과 : False
점수 입력하세요:  60
대상자 아닙니다.
다음 기회를...
수고하셨습니다.
```

### if ~ elif ~ else 조건문
- 여러가지 상황을 고려하는 조건문
```
조건
 └ case1 - 명령문1
 └ case2 - 명령문2
 └ case3 - 명령문3
```
#### (예시)
- 성적에 따라 A, B, C학점 부여하기
  - A : 90이상
  - B : 80이상 (90미만)
  - C : 70이상 (80미만)
 ```
score = int(input("학점을 입력하시오: "))

if score >= 90:
    print("A학점")
elif score >= 80:
    print("B학점")
elif score >= 70:
    print("C학점")
else:
    print("재수강")
- 결과
학점을 입력하시오:  100
A학점
```

### 파이썬 반복문
- while 반복문
```
- while 기본 문법

while 조건문 :
      명령문1
  명령문2
```
- for 반복문
```
- for 기본 문법

for 변수 in range (시작값, 종료값, 증가값) :
   명령문1
명령문2
```

#### (예시 - while 반복문)
```
n = 1
while n <= 5:
    print("파이썬 재밌네")
print("파이썬 쉽네")
```
- → 5번만 출력되도록 변경
- → 1초 간격으로 출력되도록 변경
```
# 변경 후 코드
import time

n = 1
while n <= 5:
    print("파이썬 재밌네")
    time.sleep(1)
    n += 1

print("파이썬 쉽네")
- 결과
파이썬 재밌네
파이썬 재밌네
파이썬 재밌네
파이썬 재밌네
파이썬 재밌네
파이썬 쉽네
```
#### (예시 - while | 숫자세기 : 1씩 증가하기)
```

```

### break, continue, else 키워드
- 반복문에서 활용되는 특별한 키워드
- 흐름을 제어하는 역할

#### break 키워드
```
for x in range(1, 10, 1):
   print(x)
```
- → 1부터 10까지 출력하기
- → 5에서 break 적용하여 반복 종료하기
```
# 변경 후 코드
for x in range(1, 11, 1):
    print(x)
    if x % 5 == 0:
        break
- 결과
1
2
3
4
5
```
#### continue 키워드
```
for x in range(1, 11, 1):
   print(x)
```
- → 3의 배수는 넘기기
```
for x in range(1, 11, 1):
    if x % 3 == 0:
        continue
    print(x)
- 결과
1
2
4
5
7
8
10
```
