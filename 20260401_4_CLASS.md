### 적정 체중 계산 함수
- 남자의 표준 체중(kg) = 키(m) X 키(m) X 22
- 여자의 표준 체중(kg) = 키(m) X 키(m) X 21
```
def gram(b, g):
    man_kg = b * b * 22
    lady_kg = g * g * 21

    print("남자의 표준 체중 : %.1f" % man_kg, "kg")
    print("여자의 표준 체중 : %.1f" % lady_kg, "kg")

gram(1.75, 1.50)

- 결과
남자의 표준 체중 : 67.4 kg
여자의 표준 체중 : 47.2 kg
```

#### 원의 넓이와 둘레 구하기
```
def area(r):
    pi = 3.14
    cir_len = 2 * r * pi
    cir_area = r * r * pi
    print("반지름이 %3d인 원의 둘레는 %8.2f입니다." % (r, cir_len))
    print("반지름이 %3d인 원의 넓이는 %8.2f입니다.\n" % (r, cir_area))

area(10)
area(15)
area(20)

- 결과
반지름이  10인 원의 둘레는    62.80입니다.
반지름이  10인 원의 넓이는   314.00입니다.

반지름이  15인 원의 둘레는    94.20입니다.
반지름이  15인 원의 넓이는   706.50입니다.

반지름이  20인 원의 둘레는   125.60입니다.
반지름이  20인 원의 넓이는  1256.00입니다.
```

### 함수 활용문제[매개변수가 있는 경우]

```
문제 2 
a = int(input("a정수를 입력하세요 : "))
b = int(input("b정수를 입력하세요 : "))

def call(a, b):
    plus = a + b
    minus = a - b
    axb = a * b

    print("%d + %d = %d" % (a, b, plus))
    print("%d - %d = %d" % (a, b, minus))
    print("%d * %d = %d" % (a, b, axb))

    if b != 0:
        annb = a / b
        print("%d / %d = %.2f" % (a, b, annb))
    else:
        print("0으로 나눌 수 없습니다")

call(a, b)

- 결과
a정수를 입력하세요 :  3
b정수를 입력하세요 :  4

3 - 4 = -1

3 / 4 = 0.75
```
