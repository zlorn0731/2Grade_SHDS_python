### Numpy
- Numpy는 데이터분석에서 핵심적인 파이썬 라이브러리
  과학적 계산을 위한 파이썬 외부 라이브러리
  데이터분석, 시각화, 머신러닝 등의 작업에 필수적
  벡터, 행렬 등의 자료구조 및 연산 지원
- 제공하는 기능들
  - 통계 함수들 : 최대, 최소, 평균, 중간값, 분산, 표준편차, n분위수
  - 수학 함수들 : 삼각함수, 로그함수 등
  - 벡터 및 행렬 연산 : 행렬의 곱, 역행렬, 전치행렬
  - 공학수학, 선형대수학 등
- Numpy의 장점
  - 데이터를 생성할 수 있음
    - 특정 패턴의 수열, 랜덤 수, 특정 분포에 근거한 데이터, 수학 함수 데이터(삼각 함수 등)
  - 많은 데이터를 쉽고 빠르게 처리할 수 있음
    - 리스트로 하는 것보다 수행 속도가 훨씬 빠름
    - 코드도 훨씬 짧음
  - 복잡한 연산을 수행할 수 있음
    - 통계, 선형 대수(행렬 연산 등), 푸리에 연산 등

### Numpy가 왜 필요할까? (점수 평균)
```
문제
5명의 학생의 중간고사, 기말고사 성적을 평균내어 최종 점수를 구해보자.
- 중간고사, 기말고사를 동일한 비중으로 평균내어 최종 성적 선출
- 중간고사 40%, 기말고사 60%의 비중으로 최종 성적 산출
리스트를 활용하여 시험 성적을 저장하면 어떤 문제가 있는지 생각해보자.
Numpy를 활용하면 훨씬 편리하게 계산할 수 있다.

중간고사 성적 : [90, 80, 70, 60, 50]
기말고사 성적 : [80, 70, 60, 50, 40]

계산하기
중간고사와 기말고사를 평균한 결과를 구하시오. [85, 75, 65, 55, 45]
전체적으로 +7의 가산점을 적용하시오. [92, 82, 72, 62, 52]
```
```
import numpy as np
a = [90, 80, 70, 60, 60]
b = [80, 70, 60, 50, 40]

mid = np.array(a)
fin = np.array(b)

score = (mid + fin) / 2
realscore = score + 7
print("평균 점수: ", score)
print("가산점 적용 후: ", realscore)

- 결과
평균 점수:  [85. 75. 65. 55. 50.]
가산점 적용 후:  [92. 82. 72. 62. 57.]
```

#### 기본 통계 함수들
- 합(Sum) : np.sum
- 평균(Average) : np.mean
- 분산(Variation) : np.var
- 표준편차(Standard Deviation) : np.std
```
문제
5명의 학생들에 대한 성적을 입력받아 데이터를 생성
생성한 성적에 대하여 아래의 통계, 수학 계산을 수행
합, 평균, 분산, 편차
```
```
import numpy as np

scores = []
for i in range(5):
    val = int(input("%d번 성적입력 -> " % (i+1)))
    scores.append(val)

print("\n입력된 전체점수")
print(scores)

sumR = np.sum(scores) # 합
aveR = np.mean(scores) # 평균
varR = np.var(scores) # 분산
stdR = np.std(scores) # 표준편차

print("합계: %10d" % sumR)
print("평균: %10.2f" % aveR)
print("분산: %10.2f" % varR)
print("편차: %10.2f" % stdR)

- 결과
1번 성적입력 ->  90
2번 성적입력 ->  80
3번 성적입력 ->  100
4번 성적입력 ->  70
5번 성적입력 ->  80

입력된 전체점수
[90, 80, 100, 70, 80]
합계:        420
평균:      84.00
분산:     104.00
편차:      10.20
```

#### Numpy 배열 - 데이터 분석에 사용하는 데이터형
- 리스트
  - 파이썬 데이터 분석에서 가장 많이 사용하는 기본 데이터형
  - 다양한 값을 연속으로 저장하는 것
- 넘파이 배열
  - 넘파이 라이브러리에 제공하는 ndarray 형태의 데이터형
  - ndarray는 n-dimension array(n-차원 배열)이라는 의미
  - 데이터 분석 처리에 주로 사용
- 데이터프레임
  - 판다스 라이브러리에서 제공하는 데이터형
  - 데이터 전처리에 주로 사용

- 리스트와 넘파이 배열
- 
| 리스트 | 넘파이 |
|--------|--------|
| 동작 속도가 느림 | 동작 속도가 빠름 |
| 쉼표로 요소 구분 | 공백으로 요소 구분 |
| 서로 다른 데이터형 사용 가능 | 한 가지 데이터형만 사용 가능 |
- 주요 넘파이 배열 속성
- 
| 속성 | 설명 | 속성 | 설명 |
|-----|------|------|------|
| dtype | 데이터형 표시 | T | 배열의 행/열 방향 변경 |
| shape | 배열이 몇 행 몇 열인지 표시 | size | 배열의 원소 수 표시 |
| ndim | 배열의 차원 표시 | nbytes | 배열의 byte 표시 |
| flat | 배열의 값 초기화 |
- arange : 특정 수열을 만들어 넘파이 배열을 생성하는 함수
- reshape : 배열의 차원 구조를 설정하는 함수
```
예시:
d = np.arrange(12).reshape(3, 4)
print(d, d.shape)
```
- dtype
  - 배열의 데이터형 표시 # 컴퓨터마다 다름
- ndim
  - 배열의 차원을 나타냄
  - 1차원은 1, 2차원은 2, 3차원은 3으로 표시
- T
  - 배열의 행과 열 방향을 변경하여 바꿔서 표시(전치 행렬)
- size
  - 배열 요소의 개수 표시
- nbtypes
  - 배열 요소의 총 byte 수 표시
- flat
  - 배열 요소의 값 초기화
 
#### n차원 넘파이 배열
```
예시:
(a) 1차원 배열: (x,)
(b) 2차원 배열: (x,y) / 행렬
(c) 3차원 배열: (x, y, z) / (면, 행, 열)
그림 Numpy 라이브러리 41pg 참고
```
- 1차원 넘파이 배열 생성
  - 하나의 행 또는 열만으로 이루어진 형태(Rank1으로 표현)
  - shape 함수를 사용하여 구조를 출력하면 (n,) 형태로 표시
  - arrange 함수를 사용하는 방법
    - 특정 수열을 만들어 넘파이 배열 생성
    - 넘파이의 arrange 함수와 파이썬의 range 함수는 형식이 동일하다고 해도 무방
    - arrange(시작값, 종료값, 증가값)
    - 시작값부터 종료값보다 작은 수(즉, 종료값 -1)까지 증가값만큼 터지는 배열 생성
    - 시작값은 생략 시 0, 증가값은 생략 시 1 반환
- 2차원 넘파이 배열 생성
  - 행과 열이 모두 있는 형태(Rank2로 표현)
  - 2행 3열의 배열 생성
    - np.array 함수만 사용하여 배열을 생성할 수 있음
  - 3행 2열의 배열 생성
    - np.array 함수만 사용하거나
      - np.array 함수에 arrange 함수를 함께 사용해서 생성할 수 있음
    - shape 함수로 배열의 행과 열의 개수를 각각 확인 가능
- 3차원 배열 생성
  - 행과 열로 이루어진 2차원 배열이 여러 개 있는 형태(Rank3로 표현)
  - 2차원 배열보다 열고 닫는 대괄호가 하나씩 더 있는 형태

#### 배열의 형태 변경
- 배열의 구조 변경하기 : reshape 함수
  - 배열 구조 변경 시 reshape 함수 사용
- 행과 열의 방향 변경 가능
- 1차원 배열을 2차원 배열로, 2차원 또는 1차원 배열을 3차원 배열로 변경 가능
- 배열의 면 수, 행 수, 열 수 중 하나를 자동으로 조정하기 원할 때는 -1로 설정
- 배열명 .reshape(행, 열)
- 배열명 .reshape(면, 행, 열) 형태로 설정하면 배열의 차원 변경 가능
- 행 수를 -1로 설장하고 열 수를 지정하면, 열의 개수는 지정한 값으로 맞추고
  행 수는 전체 원소 수에 맞춰 표시(3차원 배열도 동일 함)
