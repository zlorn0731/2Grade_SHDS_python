### 배열의 형태 변경
- 다차원 배열을 1차원 배열로 평탄화하기 : flatten 함수
```
f = a.flatten
print(f, f.shape)
```
- 배열의 방향 변경하기
  - 배열명[::-1] - 행 순서를 반대로 변경 가능, 행을 기준을 배열을 뒤집어 표시
  - 배열명[:,::-1] - 열 순서를 반대로 변경 가능, 열을 기준으로 배열의 좌/우를 뒤집어 표시
  - 배열명[::-1,::-1] - 배열의 상하좌우가 모두 거꾸로 변경되어 표시
  - 배열명.transpose() - 배열의 행과 열 방향을 변경하여 표시
  - 배열명.T - transpose 함수를 사용한 설정과 마찬가지로 배열의 방향을 변경하여 표시
