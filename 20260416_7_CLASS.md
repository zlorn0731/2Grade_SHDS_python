# 👨‍💻리눅스와 유닉스

## 6장 쉘 스크립트(shell script)

### 쉘 스크립트
- 리눅스 자동화를 위한 가장 강력한 도구
  즉, 명령어를 자동으로 실행하는 프로그램
```
Bash 스크립트 작성 및 실행 과정

(1) 에디터를 사용하여 Bash 스크립트 파일을 작성한다
  #! = "이 스크립트의 실행 엔진"
  #!/bin/bash : 이 스크립트를 bash로 실행

  file name: name.bash
  #!/bin/bash
  name="Kim"
  echo &name

(2) chmod를 이용하여 실행 모드로 변경한다
  $ chmod +x name.bash

(3) 스크립트 이름을 타입핑하여 실행한다
  $ ./name.bash
```

### 기본 변수(문자열)
- 하나의 값(문자열)만을 저장할 수 있는 변수, 따옴표 없어도 가능
```
$ 이름=단어
$ city=seoul
```
- 변수의 값 사용
```
$ echo $city

- 결과
seoul
```
- 변수에 어느 때나 필요하면 다른 값을 대입
```
$ city=pusan
```
- 한 번에 여러 개의 변수를 생성
```
$ country=korea city=seoul
```
```
$ country=Korea city=Seoul
$ echo $country $city

- 결과
Korea Seoul

공백이 있으면 반드시 "" 필요
$ address="Seoul City Gangnam Gu"
```

### 숫자 변수(정수)
- 실제로는 문자열이지만 연산할 때 숫자로 처리
- 숫자 타입 따로 없음 → $(())로 계산할 때 숫자 취급
```
#!/bin/bash
a=10
b=5
echo $((a+b))
```

### 정수 변수(실수 출력)
- Bash는 실수 연산을 직접 지원하지 않는다
  → 외부 도구를 사용해야 한다 (bc, awk)
```
echo $((5/2))
결과는 2이며 2.5아님
Bash는 정수 연산만 지원

echo "scale=2; 5/2" | bc
결과 : 2.50

#!/bin/bash
a=5
b=2

result=$(echo "scale=2; $a/$b" | bc)
echo $result
```

### 배열
- 한 변수에 여러 개의 값(문자열)을 저장할 수 있는 변수
```
$ 이름=(단어리스트)
$ city=(seoul busan daegu)

#!/bin/bash
arr=(apple banana cherry)
echo ${arr[0]}
echo ${arr[1]}
```
- 
| 리스트 사용 | 의미 |
|------------|------|
| ${city[i]} | 리스트 변수 city의 i번째 원소 |
| ${city[*]} | 리스트 변수 city의 모든 원소 |
| ${city[@]} | 리스트 변수 city의 모든 원소 |
| ${#city[*]} | 리스트 변수 city내의 원소 개수 |
| ${#city[@]} | 리스트 변수 city내의 원소 개수 |
```
리스트 변수 사용

$ echo ${city[*]}

- 결과
seoul busan daegu

$ echo ${city[1]}

- 결과
busan
```
```
리스트의 크기

$ echo ${#city[*]} // 리스트의 크기

- 결과
3

$ echo ${city[3]}
```
```
리스트 변수에 새로운 도시 추가

$ cities=(${city[*]} anyang)
$ echo ${city[3]}

- 결과
anyang
```
