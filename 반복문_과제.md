### [while문 활용문제]
```
- 1. 1부터 10까지 더한 결과 출력하기
n = 1
sum = 0
while n <= 10:
    sum = sum + n
    n = n + 1
print("1~10까지 더한 결과: %d" % sum)
- 결과
1~10까지 더한 결과: 55
```
```
2. 1부터 100까지 더한 결과 출력하기
n = 1
sum = 0
while n <= 100:
    sum = sum + n
    n = n + 1
print("1~100까지 더한 결과: %d" % sum)
- 결과
1~100까지 더한 결과: 5050
```
```
3. 1부터 10까지 홀수만 더한 결과 출력하기 / 1 3 5 7 9

sum = 0
while n <= 10:
    if n % 2 == 1:
        sum = sum + n
    n = n + 1
print("1~10까지 홀수만 더한 결과: %d" % sum)
- 결과

```
```
4. 1부터 10까지 짝수만 더한 결과 출력하기 / 2 4 6 8 10

```

```
10. while문으로 시작수와 끝수를 입력 받아 범위 내의 짝수의 합과 홀수의 합을 표시하는 프로그램을 작성하라.

end = int(input("끝수: "))

    if n % 2 == 0:
        even_sum = even_sum + n
    else:
        odd_sum = odd_sum + n
    n = n + 1

print("홀수의 합: %d" % odd_sum)
- 결과

끝수:  10
짝수의 합: 30
홀수의 합: 25
```
