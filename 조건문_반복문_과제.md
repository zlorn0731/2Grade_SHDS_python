### [문제 3]
- 한국과 프랑스의 시차는 8시간이다. 한국이 프랑스보다 8시간 빠르다. 즉, 한국이 10시 30분이라면 프랑스는 2시 30분이다.
- 한국 시간(시간, 분)을 입력받아 프랑스 시간을 알려주는 프로그램을 작성하시오. 시간은 24시간 단위로 입력 받는다.
- 즉 오후 2시라면 14시 입력한다.
```
kor_hour = int(input("한국 시간(시): "))
kor_min = int(input("한국 시간(분): "))

france_hour = kor_hour - 8
france_min = kor_min

if france_hour <= 0:
    france_hour += 24
    
print("한국 시간: %02d시 %02d분" % (kor_hour, kor_min))
print("프랑스 시간: %02d시 %02d분" % (france_hour, france_min))
```

### [문제 7]
- 퀴즈, 중간고사, 기말고사 점수를 입력받아 최종 점수를 계산한 후 최종 점수가 70점 이상이면 PASS, 그렇지 못하면 FAIL로 안내하는 프로그램을 작성하시오.
- 각 점수는 모두 100점 만점으로 입력받으며, 퀴즈 20%, 중간고사 30%, 기말고사 50%의 비중으로 최종 점수를 계산한다.
```
quiz = int(input("퀴즈 점수: "))
first_test = int(input("중간고사 점수: "))
second_test = int(input("기말고사 점수: "))

final_score = ((quiz * 0.2) + (first_test * 0.3) + (second_test * 0.5))
print("최종 점수: %.1f" % final_score)

if final_score >= 70:
    print("PASS")
else:
    print("FAIL")
```

### [문제 9]
- 출생연도를 입력받아 나이를 계산한 후 나이에 따라 어린이, 초등학생, 중학생, 고등학생, 대학생, 직장인을 예측하는 프로그램을 만드시오.
| 나이 | 판별 내용 |
|----------|---------|
| 1 ~ 7    | 어린이   |
| 8 ~ 13   | 초등학생 |
| 14 ~ 16  | 중학생   |
| 17 ~ 19  | 고등학생 |
| 20 ~  26 | 대학생   |
| 27 ~     | 직장인   |
```
while True:
    birth_year = input("출생연도를 입력해주세요(4자리): ")
    if len(birth_year) == 4 and birth_year.isdigit():
        print("출생연도는 %s년" % birth_year)
        break
    else:
        print("출생연도는 4자리로 입력해주세요.")

from datetime import datetime
now = datetime.now()
now_old = now.year - int(birth_year) + 1
print("현재 나이는 %d살 입니다." % now_old)

if 1 <= now_old <= 7:
    print("어린이")
elif now_old <= 13:
    print("초등학생")
elif now_old <= 16:
    print("중학생") 
elif now_old <= 19:
    print("고등학생")
elif now_old <= 26:
    print("대학생")  
else:
    print("직장인")
```

### [문제 10]
- 키를 입력받아 표준 몸무게를 계산한 후 아래의 표를 기준으로 건강을 안내해주는 프로그램을 작성하시오.
- 표준 몸무게는 키에서 100을 뺀 후 0.9를 곱하여 계산하시오.
| 비율 | 판단 | 진단 |
|---------|--------|--------------------------------|
| 85% 미만 | 저체중 | 제 때에 많이 먹고 운동도 하세요. |
| 85% 이상 ~ 115% 미만 | 정상 몸무게 | 지금 체중을 잘 유지하세요. |
| 115% 이상 ~ 130% 미만 | 과체중 | 약간 살이 쪘네요. 주 2일은 운동하세요. |
| 130% 이상 | 비만 | 식사량을 줄이고 주 3일 이상 운동하세요 |
```
cm = int(input("키: "))
kg = int(input("몸무게: "))

ave_kg = (cm - 100)  * 0.9
print("표준 몸무게: %d" % ave_kg)

ratio = (kg / ave_kg) * 100
print("비율: %d" % ratio)

if ratio < 85:
    print("저체중")
    print("제 때에 많이 먹고 운동도 하세요.")
elif ratio < 115:
    print("정상 몸무게")
    print("지금 체중을 잘 유지하세요.")
elif ratio < 130:
    print("과체중")
    print("약간 살이 쪘네요. 주 2일은 운동하세요.")
else:
    print("비만")
    print("식사량을 줄이고 주 3일 이상 운동하세요.")
```

### [문제 11]
- 키(cm)와 몸무게(kg)를 입력 받아 BMI 지수를 계산한 후 BMI 값의 범위에 따라 아래의 표를 기준으로 하여 건강을 진단해주는 프로그램을 작성하시오.
- 입력을 받을 때 체중(몸무게)은 kg 단위, 신장(키)는 cm 단위로 입력받도록 한다.
- BMI = 몸무게/키^2, 체중은 kg단위, 신장(키)는 meter단위로 계산
| BMI | 진단 |
|---------|--------|
| 20 미만 | 저체중 |
| 20 ~ 25 미만 | 정상 |
| 25 ~ 30 미만 | 과체중(1도 비만) |
| 30 ~ 40 | 비만(2도 비만) |
| 40 초과 | 고도비만(3도 비만) |
```
height = int(input("키(cm)를 입력해주세요: "))
weight = int(input("몸무게(kg)를 입력해주세요: "))

height = height * 0.01

BMI = weight / (height ** 2)
print("BMI: %.1f" % BMI)

if BMI < 20:
    print("저체중")
elif BMI < 25:
    print("정상")
elif BMI < 30:
    print("과체중(1도 비만)")
elif BMI <= 40:
    print("비만(2도 비만)")
else:
    print("고도비만(3도 비만)")
```
