## ☑️ 조건문 확인 학습 문제
### [문제 1]
- 정수를 입력받아 양수인지, 음수인지, 0인지 판단하는 프로그램을 작성하시오.
```
value = int(input("정수를 입력하시오: "))

if value > 0:
    print("%d은 양수입니다." % value)
elif value < 0:
    print("%d은 음수입니다." % value)
else:
    print("0입니다.")
```

### [문제 2]
- 출생연도를 입력받아 나이를 계산하여 알려주는 프로그램을 작성하시오. "당신은 올해 23살 입니다."라는 형식을 출력하시오.
- 또한 나이가 20세 이상이면 "당신은 성인입니다."라고 안내하시오
```
birth_year = int(input("출생연도를(yyyy) 입력하시오: "))

now_old = 2026 - birth_year + 1

print("당신은 올해 %d살 입니다." % now_old)

if now_old >= 20:
    print("당신은 성인입니다.")
else:
    print("당신은 성인이 아닙니다.")
```

### [문제 3]
- 한국과 프랑스의 시차는 8시간이다. 한국이 프랑스보다 8시간 빠르다. 즉, 한국이 10시 30분이라면 프랑스는 2시 30분이다.
- 한국 시간(시간, 분)을 입력받아 프랑스 시간을 알려주는 프로그램을 작성하시오. 시간은 24시간 단위로 입력 받는다.
- 즉 오후 2시라면 14시 입력한다.
```
kor_hour = int(input("한국 시간(시): "))
kor_min = int(input("한국 시간(분): "))

france_hour = kor_hour - 8
france_min = kor_min

if france_hour <= 0:
    france_hour += 24
    
print("한국 시간: %02d시 %02d분" % (kor_hour, kor_min))
print("프랑스 시간: %02d시 %02d분" % (france_hour, france_min))
```

### [문제 5]
- 양의 정수를 입력받아 3의 배수인지 아닌지 판단해주는 프로그램을 만드시오.
- 만약 0이나 음의 정수를 입력하면 잘못된 입력이라고 안내하고 프로그램을 종료하시오.
```
import time
value = int(input("정수를 입력하시오: "))
time.sleep(1)

if value > 0:
    print("양의 정수입니다.")
    time.sleep(1)
    if value % 3 == 0:
        print("3의 배수입니다.")
    else:
        print("3의 배수가 아닙니다.")
else:
    print("잘못된 입력입니다.")
```

### [문제 6]
- 양의 정수를 입력받아 2의 배수, 3의 배수, 5의 배수 여부를 판단해주는 프로그램을 작성하시오.
- 만약 10을 입력했다면 '10은 2의 배수입니다.', '10은 3의 배수입니다.', '10은 5의 배수입니다.'라고 3줄의 문장이 출력되도록 한다.
```
value = int(input("양의 정수를 입력하시오: "))

if value > 0:
    print("양의 정수가 맞습니다.")

    if value % 2 == 0:
        print("%d은 2의 배수입니다." % value)
    else:
        print("%d은 2의 배수가 아닙니다." % value)

    if value % 3 == 0:
        print("%d은 3의 배수입니다." % value)
    else:
        print("%d은 3의 배수가 아닙니다." % value)

    if value % 5 == 0:
        print("%d은 5의 배수입니다." % value)
    else:
        print("%d은 5의 배수가 아닙니다." % value)

else:
    print("잘못된 입력입니다.")
```

### [문제 7]
- 퀴즈, 중간고사, 기말고사 점수를 입력받아 최종 점수를 계산한 후 최종 점수가 70점 이상이면 PASS, 그렇지 못하면 FAIL로 안내하는 프로그램을 작성하시오.
- 각 점수는 모두 100점 만점으로 입력받으며, 퀴즈 20%, 중간고사 30%, 기말고사 50%의 비중으로 최종 점수를 계산한다.
```
quiz = int(input("퀴즈 점수: "))
first_test = int(input("중간고사 점수: "))
second_test = int(input("기말고사 점수: "))

final_score = ((quiz * 0.2) + (first_test * 0.3) + (second_test * 0.5))
print("최종 점수: %.1f" % final_score)

if final_score >= 70:
    print("PASS")
else:
    print("FAIL")
```

### [문제 8]
- 봄(3~4월) 또는 가을(10~11월) 정도의 시기라고 가정해보자.
- 현재 기온을 입력받아 입고 나갈 옷을 추천해주는 프로그램을 작성하시오.
```
| 온도 | 날씨안내 | 옷 추천 |
|---------|--------------------------|-----------------------|
| ~ 10    | 날씨가 춥습니다.          | 두꺼운 외투를 챙기세요. |
| 11 ~ 15 | 날씨가 선선하네요.        | 가벼운 외투를 챙기세요. |
| 16 ~ 19 | 생활하기 좋은 기온입니다. | 가볍게 입으세요.        |
| 20 ~    | 덥게 느껴지는 날씨입니다. | 반팔티도 괜찮습니다.    |
```
```
temp = int(input("현재 기온: "))

if temp <= 10:
    print("날씨가 춥습니다.")
    print("두꺼운 외투를 챙기세요.")

elif temp <= 15:
    print("날씨가 선선하네요.")
    print("가벼운 외투를 챙기세요.")

elif temp <= 19:
    print("생활하기 좋은 기온입니다.")
    print("가볍게 입으세요.")
elif temp >= 20:
    print("덥게 느껴지는 날씨입니다.")
    print("반팔티도 괜찮습니다.")
```
